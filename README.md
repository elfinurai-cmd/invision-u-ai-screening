# AI Candidate Screening (inVision U)

## Описание
Прототип системы первичного отбора кандидатов.

Система:
- оценивает кандидатов
- выдает score
- дает рекомендацию
- объясняет результат

## Запуск
pip install -r requirements.txt
streamlit run app.py

## Функции
- анализ анкеты
- ranking кандидатов
- explainable AI

## Важно
AI НЕ принимает решение — только помогает комиссии.
